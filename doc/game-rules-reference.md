# 遊戲機制參考 — 完整規則整理

> 整理日期：2026-02-25
> 用途：龍伯事典核心玩法設計之桌遊機制參考

---

## 一、Cthulhu: Death May Die — 完整規則

### 基本資訊

- **出版**：CMON (Cool Mini Or Not)
- **設計師**：Rob Daviau, Eric M. Lang
- **人數**：1-5 人合作
- **時間**：60-120 分鐘
- **類型**：合作恐怖、模組化版圖、骰子驅動

### 配件

- 17 張雙面地圖板塊（tiles）
- 調查員面板（血量/壓力/理智軌）
- 黑色基礎骰 + 綠色加成骰（四面：成功/遠古印記/觸手/空白）
- 每劇本：8 張神話卡、15 張發現卡、1 張劇本卡、敵人模型與 token
- 舊日支配者召喚軌

### 設置

依劇本卡背面指示排列板塊、放置敵人與 token、洗牌神話卡與發現卡堆、調查員放置起始位置。

### 回合結構（每位玩家）

1. **執行 3 個行動**（可重複選擇同一行動）
2. **抽神話卡** → 觸發事件
3. **調查或戰鬥** → 安全則抽發現卡；有敵人則被攻擊
4. **回合結束** → 結算火焰傷害、舊日支配者推進

### 行動類型

| 行動 | 規則 |
|------|------|
| **移動（Run）** | 移動最多 3 格，只能通過雙向箭頭通道。離開有敵人的格子時敵人跟隨。踩到火焰 token 會著火 |
| **攻擊（Attack）** | 選一個敵人擲骰，成功＝傷害。觸手＝扣理智。殺死敵人移除模型 |
| **休息（Rest）** | 僅限安全格。回復最多 3 點（血量+壓力合計）。理智不可回復 |
| **交易（Trade）** | 同格調查員可自由交換道具與同伴 |
| **劇本行動** | 依劇本卡指示的特殊行動 |

### 板塊與空間

- 每張板塊含 1-3 個空間，以牆壁與通道定義
- 移動必須兩側都有通道箭頭才能通行
- 樓梯/隧道 token：兩個擁有相同 token 的空間視為相鄰
- 同一空間無人數上限
- 三色傳送門（紅/黃/藍）＝ 敵人生成點

### 骰子系統

- 黑骰（基礎 3 顆）+ 綠骰（技能/卡牌加成）
- **成功** ＝ 命中
- **遠古印記** ＝ 情境效果
- **觸手** ＝ 扣 1 理智（立即結算）
- **空白** ＝ 無效果
- **可消耗壓力重擲任意顆骰，無次數限制**

### 發現卡（Discovery Cards）

安全空間抽取，可能獲得：
- **同伴（Companion）**：提供技能加成或吸收傷害
- **道具（Item）**：各種效果
- **狀態（Condition）**：負面效果，不可交易

### 神話卡（Mythos Cards）

每回合抽一張：
- 召喚符號 → 棄牌堆累計 3 個召喚符號時，舊日支配者在召喚軌推進一格
- 特殊效果（移動敵人、修改規則等）
- 敵人生成指示

### 瘋狂機制

- 擲出觸手 → 理智軌右移一格
- 達到瘋狂閾值 → 翻開瘋狂卡，獲得永久效果 + 技能升級
- 舊日支配者**尚未現身**時瘋狂 ＝ 立即出局（可能導致敗北）
- 現身後瘋狂 ＝ 不出局但有負面效果

### 舊日支配者

- 召喚軌推進 → 各階段有「揭示效果」（一次性）和「持續效果」
- 每階段有獨立血量，超額傷害不延續到下一階段
- **儀式未被破壞前無法攻擊舊日支配者**

### 火焰

- 離開有火焰 token 的格子 → 火焰附著到調查員
- 回合結束時：每個火焰 token 擲 1 骰，成功＝受傷，觸手＝扣理智
- 結算後移除所有火焰 token

### 勝負條件

- **勝**：破壞儀式後擊敗舊日支配者所有階段
- **敗**：全員死亡/瘋狂（現身前任一人瘋狂即敗）、召喚軌推到第 8 格

### Fear of the Unknown 擴充新增機制

- **未知怪物（Unknown Monsters）**：隨機抽取怪物卡加入任何劇本，作為額外威脅與難度調節器
- **遺物（Relics）**：開局選取神秘道具，提供持續性能力，在瘋狂值達第三階段後增強

## 二、Beacon Patrol — 完整規則

### 基本資訊

- **出版**：Pandasaurus Games
- **設計師**：Torben Ratzlaff
- **人數**：1-4 人合作
- **時間**：約 30 分鐘
- **類型**：合作、板塊拼放（tile-laying）、探索

### 概念

玩家扮演海巡隊長，在北海逐張拼放板塊建構海岸地圖、確保燈塔與浮標安全。

### 設置

- 放置海巡總部板塊（HQ）在桌面中央
- 所有船放在 HQ 上
- 洗牌所有板塊形成抽牌堆
- 每人抽板塊（1-3 人抽 3 張，4 人抽 2 張）
- 每人獲得螺旋槳 token（單人 4 個，多人依規則）

### 回合結構

1. **放置板塊**（最多 3 張 / 4 人局 2 張）
2. **使用螺旋槳移動**（選用）
3. **交換板塊**（選用，每回合限 1 次，與其他玩家交換 1 張手牌）
4. **棄掉未放的板塊，補抽新牌**

### 板塊放置規則

| 規則 | 說明 |
|------|------|
| **方向** | 每張板塊有黑色箭頭，必須朝上放置（固定方向） |
| **位置** | 只能放在船所在板塊的正交方向（上下左右，不可對角） |
| **邊緣匹配** | 水接水、陸接陸，所有相鄰邊必須吻合 |
| **水路連通** | 新板塊必須透過水邊與船所在板塊相連 |
| **船自動移動** | 每放一張板塊，船立即移到該板塊上 |

### 移動

- 翻轉螺旋槳 token（藍→紅）＝ 移動 1 格到相鄰水路連通的板塊
- 棄掉手中未放的板塊 ＝ 每張移動 1 格
- 回合結束時螺旋槳 token 重置為藍色

### 計分

只有**完全被包圍的板塊**（四面都有相鄰板塊）才算「已探索」並得分：

| 板塊類型 | 分數 |
|---------|------|
| 燈塔（Lighthouse） | 3 分 |
| 浮標（Buoy） | 2 分 |
| 一般板塊 | 1 分 |
| 碼頭（擴充） | 1 + 每個相鄰房屋 1 分 |
| 風車（擴充） | 1 + 每個相鄰開放海域 1 分 |

### 結束條件

所有板塊放完或棄完 → 計算總分 → 對照評等表。

### 單人變體

- 4 個螺旋槳 token
- 每回合可保留 1 張板塊不棄，但補抽 2 張（補償無法交易）

### 數位版額外模式

- **探索模式**：探索可獲得額外板塊，持續到無法得分為止
- **創意模式**：無限板塊與移動，自由拼
- **桌遊模式**：忠實還原實體桌遊規則（含 Ships & Shores 擴充）

## 三、兩款遊戲對龍伯事典的啟發

| 機制 | Cthulhu DMD | Beacon Patrol | 龍伯可用方向 |
|------|-------------|---------------|-------------|
| **地圖生成** | 劇本預設排列 | 玩家逐張拼放 | → **翻牌拼放**，每張是一段地層 |
| **邊緣匹配** | 箭頭通道 | 水接水陸接陸 | → 地形類型匹配（岩壁/通道/水脈） |
| **探索驅動** | 發現卡 15 張 | 四面包圍才得分 | → 翻開的牌附帶敘事片段 + 線索 |
| **未知性** | 神話卡/未知怪物 | 抽牌池隨機 | → 抽到什麼地形就是什麼，不可預測 |
| **持續性狀態** | 瘋狂累積不可逆 | — | → 地層越深，累積的異變/亢退效果越重 |
| **難度曲線** | 召喚軌倒數壓力 | 板塊用完就結束 | → 越深處的牌池越危險 |
| **敘事層** | 發現卡＝遭遇事件 | — | → 每張地形牌附帶 AI 生成的短敘事 |
| **同伴/道具** | 發現卡取得 | — | → 武學碎片、古物、錄事工具 |
| **遺物系統** | 持續增強道具 | — | → 鰲骨遺物，隨深入增強但有代價 |
| **計分邏輯** | 擊敗 Boss | 完全包圍＝探索完成 | → 拼出完整區域＝解鎖該區線索/獎勵 |
